FRONT PAGE

This front page has six diferent HTML elements in its body as shown below:

  • Title
  • Headings
  • Paragraphs
  • Nested Elements
  • Line Breaks
  • HTML Comments

The six elements are:

-A Title shows the overall name of the website. The title appears in the browser's tab. HTML titles are written in title tags as: "< title > This is a Title < /title >".

-Headings show the appearance sizes of the HTML contents that have not yet been styled. The range from heading 1 to heading 6.

-Paragraphs are introduced using the paragraph tags: < p > The paragraph content is written here. < /p >.

-Nested Elements have element(s) within another element. In this case several HTML tags are inter-used.

- Line breaks in HTML introduce a new line when an element's content is too long to fit in the window. We use the < br > tag to denote "break" the line.

- A comment in HTML assists in code follow up while coding. Comments can show the "meaning" of the following code without the texted characters being considered as a code. < !-- We comment this way in HTML --! >
